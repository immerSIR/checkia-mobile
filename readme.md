## 🎯 Présentation

**Check-IA** est une application mobile de vérification d'informations (fact-checking) pensée pour les utilisateurs francophones d'Afrique subsaharienne. Elle permet de vérifier en quelques secondes :

- 📝 **Textes** — articles, publications, rumeurs
- 🔗 **URLs** — liens d'articles avec aperçu automatique
- 🖼 **Images** — détection de deepfakes et vérification de contenu
- 🎵 **Audio** — transcription et détection de voix IA

# Support

Use GitHub issues for reproducible bugs, documentation problems, and scoped feature requests.

Before opening an issue:

- Search existing issues.
- Confirm the problem still happens on the latest `develop` branch.
- Run `npm install` and restart Expo with a cleared cache if the problem is local.

For security issues, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

For general questions, include enough context for maintainers to answer:

- Operating system.
- Node.js and npm versions.
- Expo target, such as Android, iOS, web, or Expo Go.
- Relevant logs or screenshots.
- Steps already tried.
